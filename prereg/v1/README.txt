README — Pre-Pilot Preregistration

This folder contains the pre-pilot preregistration for the project
“Sequence-Dependent Distortions in Betting”

File:
- sb-prereg-v1.pdf

Status:
- Pre-pilot preregistration
- Finalized prior to any pilot or outcome-informative data collection
- No empirical results from the main laboratory study were observed or used

Scope:
- Specifies all confirmatory and exploratory analyses for the main laboratory experiment
- Confirmatory inference is restricted to the False–Negative treatment
- Exploratory analyses, robustness checks, and a classroom-based sanity-check study
  are explicitly designated as non-confirmatory

Notes:
- Any future changes will be documented via versioned preregistration updates
